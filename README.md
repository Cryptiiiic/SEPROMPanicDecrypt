# SEPROM Panic Decrypt

## Dependencies:
`pip3 install pycryptodome`

## Usage:
`python3 main.py <SoC> <SEPROM Panic Bytes>`

## Example:

## Supported SoC's:
* a8
* a8x
* a9(Samsung)
* a9x
* a9(TSMC)
* S4/S5 Watch
* S1p/S2 Watch/T1 Chip
* a10
* a10x
* T2 Chip
* a11
* a12
* a12z/x
* a13
* a14
* M1
* a15
* a16(A0)
* a16(B1)
* M2
* a17
* M3 Max(Binning 1)
* M3 Ultra
* M3 Max(Binning 2)

## Unsupported SoC's:
* S9/S10 Watch
* M3
* M4
